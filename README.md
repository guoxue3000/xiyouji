## 关于西游记
《西游记》，是中国古代明朝的第一部浪漫主义章回体长篇神魔小说，全书58.5万字（世德堂本），共100回，中国四大名著之一、四大奇书之一。
成书于16世纪明朝中叶，一般认为作者是明朝的吴承恩。书中讲述唐三藏与徒弟孙悟空、猪八戒和沙悟净等师徒四人前往西天取经的故事，表现了惩恶扬善的古老主题，兴佛灭道的代表作，
也有观点认为《西游记》是暗讽权力官场的讽刺小说。
